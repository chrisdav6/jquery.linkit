# jQuery LinkIt Plugin #
Link it allows you to add a link to HTML elements

## Version 0.2.0 ##

## Usage ##

	$("span").linkit({
		href: "http://google.com",
		text: "this ding dang text ya dingus",
		target: "_blank"
	});
